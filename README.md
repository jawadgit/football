# Football Team 
- Football tema and players project built in symfony
# Requirements
- PHP 7.3 version or greater https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/7.4.30/
- composer 2.5.8 https://getcomposer.org/download/
- symfony CLI 5.5.6 https://symfony.com/download

# Package used for development
- Pagination  https://github.com/KnpLabs/KnpPaginatorBundle
 
# Installation

- clone repo
- create database namely teamplayers
- import database file placed on root namely teamplayers.sql
- update .env database name and password accordingly
- For vendor dependencies run command `composer install`
- For starting project run command `symfony server:start`
- For Test run command ` symfony php bin/phpunit tests/Controller/`
